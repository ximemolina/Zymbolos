# Zymbolos
Proyecto de **Compiladores e Intérpretes**

---
### Uso
Escribir en la terminal el siguiente comando
```bash
python main.py ejemplos/[nombreDeArchivoAProcesar].txt
```

---
### Pruebas
Correr todas las suites en resumen:
```bash
python -m tests.runner
```

Correr una suite específica con detalle:
```bash
python -m tests.test_hojas
python -m tests.test_internos
```
